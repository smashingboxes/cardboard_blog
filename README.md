# Cardboard Blog

## Usage

```ruby
#Gemfile
gem 'cardboard_cms', git: 'git@github.com:smashingboxes/cardboard.git'
gem 'cardboard_blog', git: 'git@github.com:smashingboxes/cardboard-blog.git'

```

```bash
rails generate cardboard:blog:install
rake db:migrate

```
