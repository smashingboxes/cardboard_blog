# Cardboard Blog

## Usage

```ruby
#Gemfile
gem 'cardboard_cms', git: 'git@github.com:smashingboxes/cardboard.git'
gem 'cardboard_blog', git: 'git@github.com:smashingboxes/cardboard_blog.git'

```

```bash
rake cardboard_blog:install:migrations
rake db:migrate

```
