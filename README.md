# Cardboard-Blog

## Usage

```ruby
#Gemfile
gem 'cardboard-cms', git: 'git@github.com:smashingboxes/cardboard.git', require: 'cardboard'
gem 'cardboard-blog', git: 'git@github.com:smashingboxes/cardboard-blog.git'

```

```bash
rails generate cardboard:blog:install
rake db:migrate

```
